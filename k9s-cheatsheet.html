⚙ Configuration

config.yaml (k9s.yaml) — 메인 설정 Config
$XDG_CONFIG_HOME/k9s/config.yaml
k9s의 핵심 설정 파일. UI, 로거, 리소스 임계값, 셸 파드, 이미지 스캔 등 전체 동작을 제어합니다.

apiServerTimeoutstring2m0sK8s API 요청 타임아웃

readOnlyboolfalse읽기 전용 모드
noExitOnCtrlCboolfalseCtrl+C로 종료 방지

aliases.yaml — 커스텀 별칭 Alias
$XDG_CONFIG_HOME/k9s/aliases.yaml
GVR (Group/Version/Resource) 형식으로 커스텀 리소스 별칭을 정의합니다.
GVR 형식 규칙
형식예시
group/version/resourceapps/v1/deployments
version/resourcev1/secrets
resource-onlydeployments
subresource (:)v1/pods:log
로드 우선순위 (낮 → 높)
1. k9s 내부 하드코딩 (h, q, a, ctx, pf, sd, x)
2. K8s API 동적 short name (po, svc 등)
3. 전역 aliases.yaml
4. 컨텍스트별 clusters/<cluster>/<ctx>/aliases.yaml
제약: Define() 가드로 먼저 등록된 alias는 오버라이드 불가. 컨텍스트 파일은 신규 alias 추가 전용.
aliases:
  dp: deployments
  sec: v1/secrets
  jo: jobs
  cr: clusterroles
  np: networkpolicies
hotkeys.yaml — 커스텀 단축키 Hotkey
$XDG_CONFIG_HOME/k9s/hotkeys.yaml
컨텍스트별: $XDG_DATA_HOME/k9s/clusters/<cluster>/<context>/hotkeys.yaml
필드타입기본값설명
shortCutstring(필수)tcell 키 이름
descriptionstring""헬프 화면 표시 텍스트
commandstring""실행 커맨드, 환경변수 치환 지원
overrideboolfalsetrue면 내장 단축키 강제 대체
keepHistoryboolfalsetrue면 네비게이션 히스토리 유지
단독: a~z, 0~9, ?, /, space, :, -, [, ]
Ctrl: Ctrl-A~Ctrl-Z
Shift: Shift-A~Shift-Z, Shift-0~Shift-9
특수: Backspace, Delete, Enter, Tab, Up, Down, Left, Right, PgUp, PgDn, Home, End, F1~F12
$NAMESPACE $NAME $RESOURCE_NAME $RESOURCE_GROUP $RESOURCE_VERSION $CONTEXT $CLUSTER $FILTER $POD $CONTAINER $COL-<컬럼명>
치환 형식: $VAR, ${VAR}, $!VAR (불리언 반전)
override 동작: false → 충돌 시 핫키 무시 + 에러 로그. true → 기존 내장 단축키 강제 대체.

plugins.yaml — 플러그인 설정 Plugin
$XDG_CONFIG_HOME/k9s/plugins.yaml
외부 명령어를 k9s에 통합하는 플러그인 시스템. JSON 스키마 3종 제공: plugin.json, plugins.json, plugin-multi.json
필드타입기본값설명
shortCutstring(필수)키 바인딩
descriptionstring(필수)UI 힌트바 설명
scopes[]string(필수)활성화될 리소스 뷰 ("all" 또는 리소스명)
commandstring(필수)실행 바이너리
args[]string[]커맨드 인수, 환경변수 치환 지원
backgroundboolfalsetrue면 UI 벗어나지 않고 백그라운드 실행
confirmboolfalse실행 전 확인 다이얼로그
dangerousboolfalsetrue면 readOnly 모드에서 비활성화
pipes[]string[]stdout 파이프 체인
overrideboolfalse동일 단축키 기존 액션 덮어씀
overwriteOutputboolfalsebackground 실행 결과를 Flash 메시지 표시
공통 (k8sEnv): $CONTEXT $CLUSTER $USER $GROUPS $KUBECONFIG
리소스뷰 (defaultEnv): $NAMESPACE $NAME $COL-<컬럼명>
테이블뷰: $FILTER $RESOURCE_GROUP $RESOURCE_VERSION $RESOURCE_NAME
컨테이너/xray: $POD $CONTAINER
고급 치환: $!VAR (불리언 반전), ${COL-%CPU/L} (특수문자 컬럼명), ${READINESS GATES} (공백 포함)
참고: inputs 필드는 v0.50.18에 존재하지 않습니다.
plugins:

    dangerous: true

      - --image=busybox
      - -n
      - $NAMESPACE
      - $NAME
  remove-finalizers:
    shortCut: Ctrl-F
    description: "Remove Finalizers"
    scopes: [all]
    command: kubectl
    confirm: true
    dangerous: true
    background: true
    overwriteOutput: true
    args:
      - patch
      - $RESOURCE_NAME.$RESOURCE_GROUP
      - $NAME

리소스 뷰의 커스텀 컬럼 및 정렬 순서를 GVR 기반으로 정의합니다.
GVR 뷰 이름 형식
형식예시비고
version/resourcev1/pods코어 리소스
group/version/resourceapps/v1/deployments확장 리소스
...@namespacev1/pods@default네임스페이스 지정
...@patternv1/pods@ns*정규식 (우선순위 높음)
컬럼 정의 문법
형식: COLUMN_NAME:JSONPATH|FLAGS — 파싱 정규식: ^([\w\s%/-]+):?([\w\W]*?)\|?([NTWSLRH]{0,3})$
Flag의미
WWide-only (좁은 뷰에서 숨김)
SAlways-show (좁은 뷰에서도 표시)
HHide (항상 숨김)
TTime (RFC3339→상대시간 변환)
NNumber (우측 정렬, 용량 포맷)
LAlignLeft
RAlignRight
정렬 문법: sortColumn: "AGE:asc" 또는 "NAME:desc"
views:
  v1/pods:
    columns:
      - NAME
      - READY
      - STATUS
      - AGE|TR
      - IMG:.spec.containers[0].image|split(":")|.[0]
      - VERSION:.spec.containers[0].image|split(":")|.[-1]|W
    sortColumn: "AGE:desc"
  v1/pods@default:
    columns:
      - NAME
      - STATUS
      - AGE
  apps/v1/deployments:
    columns:
      - NAME
      - READY
      - UP-TO-DATE
      - AVAILABLE
      - AGE|T
    sortColumn: "NAME:asc"
  v1/services:
    columns:
      - NAME
      - TYPE
      - CLUSTER-IP
      - PORT(S)|S
      - AGE
Context-level config.yaml Context Config

컨텍스트별 독립 설정. 네임스페이스 즐겨찾기, 활성 뷰 등을 클러스터/컨텍스트 조합마다 개별 저장합니다.
필드타입기본값설명
namespace.activestring"default"현재 활성 네임스페이스
namespace.lockFavoritesboolfalsetrue면 favorites 자동 업데이트 잠김
namespace.favorites[]string[]즐겨찾기 네임스페이스 (최대 9개, LRU)
view.activestring"po"마지막 활성 뷰
참고: 네임스페이스 전환 시 자동 LRU 추가. lockFavorites: true 설정 시 수동 설정만 사용. 숫자키 0~9는 리소스 브라우저 뷰에서만 동작 (Log 뷰 시간 필터와 별도).
k9s:
  namespace:
    active: monitoring
    lockFavorites: false
    favorites:
      - default
      - kube-system
      - monitoring
      - ingress-nginx
      - cert-manager
  view:
    active: deploy
